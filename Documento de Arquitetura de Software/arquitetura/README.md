# Documento de Arquitetura de Software

Este projeto contém uma representação em HTML/CSS do Documento de Arquitetura de Software, simulando o formato de um PDF dentro do VS Code.

## Estrutura do Projeto

- `index.html` - Estrutura principal do documento
- `styles.css` - Estilos CSS para formatação similar a PDF

## Características

- ✅ Design responsivo
- ✅ Estilo similar a documento PDF/A4
- ✅ Tabelas formatadas para histórico de revisões e aprovações
- ✅ Sumário com links internos
- ✅ Caixas de conteúdo para seções de texto
- ✅ Suporte a impressão (otimizado para print)
- ✅ Visualização fiel ao documento original

## Como Visualizar

1. Abra o VS Code
2. Instale a extensão "Live Server" (opcional, mas recomendado)
3. Clique com botão direito no `index.html`
4. Selecione "Open with Live Server"

Ou simplesmente abra o arquivo `index.html` diretamente no navegador.


## Pré-requisitos

- Navegador web moderno
- VS Code (para edição)
- Conhecimentos básicos de HTML/CSS (para personalização)
